Quick Start Guide

  1. Install the extension from the Chrome Web Store or load it unpacked via chrome://extensions ‚Üí Developer mode ‚Üí Load unpacked.
  2. Navigate to www.clzschoolplein.nl. The dark theme applies automatically on first visit.
  3. Click the extension icon in your toolbar to toggle dark mode, customize colors, or reset preferences.
  4. New! Access the customization panel to modify colors, backgrounds, and styling options.
üí° Pro Tip: Pin the extension to your toolbar for quick access to customization options!

Popup & Settings

The extension popup provides quick controls to toggle dark mode and access customization options. All preferences are stored in chrome.storage.local and sync across all tabs instantly.

Troubleshooting

  • Popup shows wrong state? Click the Reset button and reload the page.
  • Custom colors not applying? Make sure you've saved your changes and refreshed the page.
  • Extension not working? Check that it's enabled at chrome://extensions and the URL matches https://www.clzschoolplein.nl/.
  • Open DevTools Console on the site to check for any error messages from content_script.js.
