CLZ Dark Mode - Help & Instructions

A lightweight Chrome extension that applies a site-specific dark theme to www.clzschoolplein.nl.

Quick start

  1. Install the extension from the Chrome Web Store (or load unpacked via chrome://extensions → Load unpacked).
  2. Open www.clzschoolplein.nl. The dark theme is applied automatically by default.
  3. Use the extension toolbar popup to toggle dark mode or reset the stored preference.

Popup & settings

The popup stores a single preference (clzDarkEnabled) in chrome.storage.local. Toggling the switch updates the setting and notifies all open tabs on the site so the change takes effect immediately.

Troubleshooting

  • If the popup shows the wrong state, try clicking Reset and reopen the site.
  • Open Chrome DevTools on clzschoolplein.nl and check the Console for errors from content_script.js.
  • Make sure the extension is enabled and the site URL matches https://www.clzschoolplein.nl/.

Privacy

This extension runs locally in your browser. It only stores a single boolean preference in chrome.storage.local and does not collect or transmit personal data.

Developer notes

Files included in the project:

  • manifest.json (MV3)
  • content/dark.css (scoped, applied via clz-dark-mode)
  • content/content_script.js (applies class & handles messages)
  • popup/popup.html, popup/popup.js

Install locally

1. Open chrome://extensions/  
2. Enable Developer mode  
3. Click "Load unpacked" → select the extension folder
